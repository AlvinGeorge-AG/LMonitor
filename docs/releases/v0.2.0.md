# LMonitor v0.2.0

Release date: 2026-04-12

## Highlights

- Windows users can now run LMonitor with the same dashboard used on Linux
- The server and frontend protocol stay unchanged because the platform split happens only inside `internal/metrics`
- Linux keeps the original `/proc`-based collector path

## Included Artifacts

- `dist/lmonitor-v0.2.0-linux-amd64`
- `dist/lmonitor-v0.2.0-linux-arm64`
- `dist/lmonitor-v0.2.0-windows-amd64.exe`

## Platform Notes

Linux:
- Full support for the existing metric set

Windows:
- CPU, per-core CPU, RAM, swap, load, uptime, network, disk throughput, root drive usage, and total process count are supported
- Runnable-process count is still Linux-specific and may show as `0`
- Some counters depend on host-provided Windows performance data and can be best-effort
