# LMonitor v0.3.0

Release date: 2026-04-13

## Highlights

- Added structured application logging with normalized severity categories
- Added Linux `journalctl` streaming integration for live system log visibility in the dashboard
- Refined dashboard UX for log readability and overall monitoring clarity

## Included Artifacts

- `dist/lmonitor-v0.3.0-linux-amd64`
- `dist/lmonitor-v0.3.0-linux-arm64`
- `dist/lmonitor-v0.3.0-windows-amd64.exe`

## Platform Notes

Linux:
- Full metrics support remains in place
- Dashboard logs include monitor logs plus available system journal lines

Windows:
- Existing v0.2.0 metric support is unchanged in this release
- Structured monitor logs are available in the dashboard log panel

Other platforms:
- Unsupported platforms continue to fail cleanly at runtime with an explicit message until a collector is added
